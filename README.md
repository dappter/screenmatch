# Screenmatch

Este é um projeto desenvolvido como parte de estudos em Java, consumindo APIs, gravando arquivos e lidando com erros.

## 📌 Sobre o projeto

O **Screenmatch** permite realizar buscas de filmes/séries via API e exibir informações tratadas no terminal. O projeto também inclui manipulação de dados e escrita em arquivos.

## 🚀 Tecnologias utilizadas

* Java
* API HTTP Client (Java)
* Biblioteca Gson
* Manipulação de arquivos
* Tratamento de exceções

## 🧠 Conceitos praticados

* Consumo de API REST com Java
* Parse de JSON
* Orientação a Objetos
* Tratamento de erros
* Serialização e gravação de dados

## ▶️ Como executar

1. Clone este repositório

```
git clone https://github.com/dappter/screenmatch.git
```

2. Abra no IntelliJ ou outra IDE Java
3. Execute a classe principal dentro de `/src/main/java`

> Certifique‑se de ter o Java 17+ instalado
